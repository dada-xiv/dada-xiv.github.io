firebase hosting

If the site-name for site-name.web.app is different from project-id, then insert the line "site": "{site name}" to firebase.json so that 
{
  "hosting": {
    "site": "{site-name}",
    "public": "public",
    "ignore": [
      "firebase.json",
      "**/.*",
      "**/node_modules/**"
    ]
  }
}
